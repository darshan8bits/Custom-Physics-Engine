Physics Simulator (SFML + CPP)

A simple interactive 2D physics simulator built using C++ and SFML.
The goal of this project is to experiment with physics concepts such as velocity, collisions, and restitution while building a small simulation engine.

Features (Current Stage):

1. Click anywhere on the screen to spawn balls dynamically

2. Configure initial x velocity

3. Configure initial y velocity

4. Configure coefficient of restitution (bounce factor)

5. Spawn multiple balls simultaneously

6. Real-time rendering using SFML

Each ball generated follows the configured physical parameters, allowing experimentation with different motion behaviors.
